Marginal Structural model approach: IPW

Here we will used lagged values of the exposure to adjust for confounders and IP weights.

Questions

The question arises, which model to prefer? I think probably GEE.

Key points:

  • Sensitivity to IP weighting = important.
  • Model specification = important.
  • Handling of zero income?
  • Need to handle missingness!
  • Need to include more confounders, probably.
  • Draw a DAG!
  • All sorts of reasons people change in househould income.
  • Probably go with GEE.
